Hi! My name is Sarah Ridley and I'm a senior in high school in North Carolina. This website is my personal portfolio, and I coded it myself. I've used tools like WordPress on other projects, but I decided to go with pure code on this website. So, everything you see is a combination of HTML, CSS, and JavaScript. I fell in love with coding in middle school, and this website has allowed me to create a tangible project that I can easily share with others. The beauty of this website is that I can also use it to share my other interests. Believe it or not, coding is not my only hobby! I also love robotics, sewing, baking, cooking, and volleyball. Conveniently, each of those areas (as well as programming) is one of the tabs on the navigation bar. I would be grateful if you took the time to explore the projects I have posted in each of those areas. They all mean a lot to me!


Also, because I built this website from scratch (with the help of many articles and forums online), there is a chance that you will find a bug. If you're willing to go the extra mile, please contact me and tell me about the problem so that I can fix it! You can find my information on the contact tab on the bottom bar.

The second photo is from when I was finishing a quilt that I sewed for an English project at the end of my junior year. The process of finishing a quilt includes sewing lines or patterns over fabric that has already been sewn together to create the background of the quilt. These seams that run along the top and the bottom give the quilt more depth and an elevated look of precision. This quilt was so much fun to make and I learned so much about the craft! You can find more about this specific quilt on the sewing tab of my website.
